# Стили MaxiBooking для гостевого дома «Палермо»

Эти 4 CSS-блока подгоняют форму бронирования и календарь MaxiBooking
под визуал сайта (палитра espresso/gold/cream/terracotta, шрифты
Cormorant Garamond + Inter, квадратные углы 4 px).

## Куда что вставлять (админка MaxiBooking → онлайн-форма)

| Файл | Поле в админке |
| --- | --- |
| `01-search-form.css` | «Дополнительные css-стили для онлайн формы» → **CSS для формы поиска** |
| `02-results-form.css` | «Дополнительные css-стили для онлайн формы» → **CSS для формы с результатами** |
| `03-floating-button.css` | **CSS для „плавающей“ кнопки на первом шаге** |
| `04-calendar.css` | «Оформление календаря» → **Дополнительные css-стили для календаря** |

Размеры фреймов (рекомендуемые):

* Форма поиска: ширина `100%`, высота `600`
* Календарь: ширина `340px`, высота `360px`

## Как это работает

MaxiBooking рендерит свою форму внутри iframe. Стили из админки
подгружаются в этот iframe — поэтому подключать `css/style.css`
сайта туда нельзя; в каждом файле мы сами объявляем нужные
CSS-переменные и импортируем шрифты с Google Fonts.

Палитра и геометрия скопированы из `css/style.css`:

* `--pl-espresso #2a160c` — основной тёмный (текст, рамки, чёрные кнопки)
* `--pl-gold #b58a4f` / `--pl-gold-2 #d8a85e` — фон CTA-кнопок и ховер
* `--pl-cream #f4e7cf` — подложка карточек / сводки / календаря
* `--pl-terracotta #b8552d` — цены и акценты
* `--pl-radius 4px` — общий радиус (как `--radius` сайта)
* Шрифты: `Cormorant Garamond` (заголовки) + `Inter` (всё остальное)

## Как обновлять

1. Поменяли что-то в `css/style.css` (например, добавили акцентный цвет) —
   синхронизируйте переменные в этих файлах.
2. Скопируйте обновлённый файл целиком в нужное поле админки MaxiBooking.
3. Сохраните, обновите страницу `/booking.html` и проверьте, что форма
   и календарь не сломались.

## Почему `!important` везде

MaxiBooking подгружает свои стили (Bootstrap 3 + jQuery UI Datepicker)
после кастомных, поэтому без `!important` правила перебиваются.
Это компромисс под чужой движок — внутри сайта мы `!important`
не используем.
