# Palermo Guest House

Static multi-page website for GitHub Pages.

## Pages
- `index.html` — homepage
- `rooms.html` — room categories
- `services.html` — advantages and amenities
- `booking.html` — MaxiBooking search form
- `results.html` — MaxiBooking results page
- `policy.html` — privacy policy
- `404.html` — fallback page

## Local preview
```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000/`.
