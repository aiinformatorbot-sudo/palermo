Политика конфиденциальности

Настоящая политика объясняет, какие персональные данные могут обрабатываться при использовании сайта гостевого дома «Палермо» и онлайн-модуля бронирования.

1. Общие положения

Сайт «Палермо» расположен по адресу https://palermo-feo.ru/ и используется для предоставления информации о проживании, условиях размещения и онлайн-бронирования.

При оформлении бронирования на странице бронирования часть данных обрабатывается в сервисе онлайн-бронирования, подключённом к сайту.

2. Какие данные могут обрабатываться

  • Фамилия, имя и контактные данные, указанные при бронировании;
  • Даты заезда и выезда, количество гостей, выбранная категория номера;
  • Технические данные, передаваемые браузером при посещении сайта;
  • Информация, необходимая для связи по номеру +7 978 833-09-64.

3. Цели обработки

  • Оформление и сопровождение бронирования;
  • Связь с гостем по вопросам проживания;
  • Уточнение условий размещения и подтверждение заказа;
  • Обеспечение корректной работы сайта и модуля бронирования.

4. Передача данных третьим лицам

Для работы онлайн-бронирования сайт использует внешний сервис бронирования. Данные, введённые в форму бронирования, передаются оператору сервиса исключительно для оформления и исполнения брони.

5. Срок хранения и защита

Персональные данные хранятся не дольше, чем это требуется для достижения целей обработки, выполнения обязательств по бронированию и соблюдения требований закона.

Для защиты информации используются разумные организационные и технические меры, предусмотренные сервисом бронирования и инфраструктурой сайта.
